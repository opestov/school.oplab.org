Кружки для школьников

В городе Кирове работают кружки для школьников, интересующихся программированием. Главное направление обучения — алгоритмы и структуры данных.

На занятиях мы:

  • Учимся думать и думаем.
  • Изучаем языки программирования. Для получения практических навыков, пишем программы.
  • Изучаем алгоритмы и структуры данных. Для лучшего понимания решаем задачи.
  • Участвуем в олимпиадах по информатике, где решаем задачи, используя ум и полученные практические навыки.
